# TorchBloom LLM Wiki

**Status:** Bootstrap. No concept pages have been authored yet.

The LLM wiki will become TorchBloom's source-grounded AI learning knowledge base. It should connect elementary-friendly intuition to real math, programming, machine learning, deep learning, transformers, AI Olympiad preparation, and paper-inspired implementation.

## Planned Tracks

| Track | Purpose |
| --- | --- |
| Math foundations | Arithmetic, functions, probability, linear algebra, calculus, and optimization. |
| Programming foundations | Python, debugging, data structures, notebooks, and computational thinking. |
| Data and probability | Tables, charts, measurement, uncertainty, sampling, and evaluation. |
| Classical ML | kNN, decision trees, regression, clustering, features, and metrics. |
| Deep learning | Tensors, gradients, neural networks, losses, training loops, and PyTorch. |
| Transformers | Embeddings, attention, sequence modeling, generation, and inspection. |
| AI Olympiad | Reasoning patterns and prerequisite knowledge for AI competitions. |
| Paper reproductions | Simplified research-paper projects and ablations. |

## Source Intake

Sources are recorded under `raw/llm-wiki/manifests/` before they are used. Raw full text is not required in git and should remain private unless license review allows it.
