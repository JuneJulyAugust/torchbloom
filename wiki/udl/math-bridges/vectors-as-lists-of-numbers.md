# Vectors As Lists Of Numbers

## Core Idea

A vector is an ordered list of numbers. UDL uses vectors when inputs or outputs
need more than one value.

## High-School Bridge

If a used car is described by age and mileage, the input can be `[age, mileage]`.
Order matters: `[age, mileage]` and `[mileage, age]` are different meanings even
if the same two numbers appear.

## Ready For UDL When

- You can read a bold symbol as "a list, not one number."
- You can explain why each position in the list must keep the same meaning.
- You can connect a two-number vector to a point on a 2D graph.
