# Coordinates And Graphs

## Core Idea

A graph turns a function into a shape. In Chapter 2, each line is a possible
linear-regression model; the training points show what the model is trying to
match.

## High-School Bridge

Each point has an `x` coordinate and a `y` coordinate. A model prediction is the
height of the model curve at that `x`. The error is the vertical distance from
the model prediction to the observed `y`.

## Ready For UDL When

- You can locate a point from `(x, y)`.
- You can identify a line's slope and intercept.
- You can describe a prediction error as a vertical distance on the graph.
