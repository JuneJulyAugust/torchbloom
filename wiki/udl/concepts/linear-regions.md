# Linear Regions

## Core Idea

A linear region is a part of input space where the same hidden units are active
and inactive, so the network behaves like one linear function there.

## Why It Matters In UDL

Counting regions helps explain why shallow networks can represent complicated
surfaces even though each hidden unit is simple.

## High-School Bridge

On a number line, a joint splits left and right regions. In two dimensions,
lines split the plane into polygonal regions.
