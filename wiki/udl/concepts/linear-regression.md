# Linear Regression

## Core Idea

Linear regression predicts an output with a line. The intercept moves the line
up or down, and the slope controls how much `y` changes when `x` changes.

## Why It Matters In UDL

UDL uses 1D linear regression as the simplest complete example of supervised
learning: model, parameters, loss, training, and testing all appear in one
visual system.

## High-School Bridge

This is the familiar equation of a line. The new move is to treat the slope and
intercept as learnable parameters rather than numbers given in advance.
