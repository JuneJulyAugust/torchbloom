# Chapter 3 Reading Guide

## What This Chapter Teaches

Chapter 3 replaces the single line from Chapter 2 with a function made from
linear pieces. ReLU hidden units create joints, and weighted sums of hidden units
create a flexible piecewise-linear prediction function.

## Before You Read

Review a line's intercept and slope, piecewise definitions, and how adding
graphs changes the resulting shape. The key mental move is: "each hidden unit is
a clipped line."
