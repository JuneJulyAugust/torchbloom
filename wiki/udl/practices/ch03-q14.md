# Ch03 Q14: Write Figure 3.11 Equations

## Prompt

Write the equations for the hidden units and outputs in figure 3.11.

## What This Tests

Whether the general network equation can be instantiated from a diagram.

## Hints

- Write one equation per hidden unit.
- Write one equation per output.

## Solution Notes

- The number of output equations matches the outputs.
