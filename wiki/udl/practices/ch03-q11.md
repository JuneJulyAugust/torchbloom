# Ch03 Q11: Count Parameters In Figure 3.6

## Prompt

Count every slope and intercept in the figure 3.6 model.

## What This Tests

Whether network diagrams can be translated into parameter counts.

## Hints

- Count input-to-hidden slopes and hidden-unit intercepts.
- Count hidden-to-output slopes and output intercepts.

## Solution Notes

Selected answer available on booklet page 9.
