# Ch02 Q03: Generative Versus Discriminative Regression

## Prompt

Rewrite the line model so it predicts input from output, derive the inverse used
for inference, and test whether this gives the same fitted predictions as the
usual discriminative model.

## What This Tests

Whether "same line-looking equation" still means the same optimization problem.

## Hints

- The discriminative model measures vertical errors in `y`.
- The generative reformulation measures errors in `x`.
- Try a tiny three-point dataset and compare fitted lines.

## Solution Notes

The selected answer says the predictions need not match; use the booklet only as
a check after deriving or coding your own comparison.

## Self-Check Rubric

- The inverse function is explicit.
- The loss is written for errors in the predicted input.
- The explanation names why the two fits can differ.
