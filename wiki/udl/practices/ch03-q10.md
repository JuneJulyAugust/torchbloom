# Ch03 Q10: Fewer Than Four Regions

## Prompt

Describe when a one-input, three-hidden-unit ReLU network could create fewer
than four linear regions.

## What This Tests

Whether "maximum possible regions" is distinguished from "regions actually
realized by a parameter setting."

## Hints

- Some joints can coincide.
- A hidden unit can have no useful slope.
- Some regions can have the same effective slope.
