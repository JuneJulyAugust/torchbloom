# Ch03 Q03: Joints And Slopes

## Prompt

Find the input locations where each hidden-unit pre-activation crosses zero,
then derive the slope in each linear region.

## What This Tests

Whether the geometry of clipped hidden units is connected to algebraic
parameters.

## Hints

- Solve each `theta_d0 + theta_d1 x = 0`.
- In a region, inactive units contribute zero slope.
- Active units contribute their hidden-unit slope times output weight.

## Solution Notes

Selected answer available on booklet page 7.

## Self-Check Rubric

- Joint positions are written in terms of theta parameters.
- Region slopes include only active hidden units.
