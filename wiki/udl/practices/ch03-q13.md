# Ch03 Q13: Activation Patterns For Seven Regions

## Prompt

For each of the seven regions in figure 3.8j, identify which hidden units are
active and inactive.

## What This Tests

Whether the one-dimensional activation-pattern idea transfers to two-dimensional
input space.

## Hints

- Each hidden unit corresponds to one boundary.
- Pick a sample point inside a region and evaluate each pre-activation sign.

## Solution Notes

Review needed against the figure.

## Self-Check Rubric

- Every region has a three-unit pattern.
- Neighboring regions differ only when crossing a hidden-unit boundary.
