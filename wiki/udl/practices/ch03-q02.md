# Ch03 Q02: Activation Patterns In Four Regions

## Prompt

For each region in figure 3.3j, mark which hidden units pass their input and
which are clipped to zero.

## What This Tests

Whether regions are understood as activation-pattern zones.

## Hints

- A ReLU unit is active when its pre-activation is nonnegative.
- Crossings of zero are the joints.
- Work left to right along the input axis.

## Solution Notes

Review needed against the figure.

## Self-Check Rubric

- Every region has a three-unit active/inactive pattern.
- The pattern changes only at joints.
