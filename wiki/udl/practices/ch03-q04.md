# Ch03 Q04: Redraw A Changed Hidden Unit

## Prompt

Change the third hidden unit as specified by the source figure and redraw how
the resulting network computation changes.

## What This Tests

Whether a hidden unit can be traced through pre-activation, ReLU, weighting, and
final output.

## Hints

- Keep the first two hidden units fixed.
- Redraw the third pre-activation first.
- Then redraw the clipped and weighted contribution.

## Solution Notes

Selected figure answer is available on booklet page 8.

## Self-Check Rubric

- Only the changed hidden unit's contribution moves.
- The final output changes where that unit is active.
