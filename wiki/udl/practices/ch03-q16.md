# Ch03 Q16: Zaslavsky Region Count

## Prompt

Use the region-count formula for hyperplanes to compare the maximum number of
regions with three hidden units and with five hidden units in two input
dimensions.

## What This Tests

Whether combinatorial region counting can be connected back to hidden-unit
boundaries.

## Hints

- For two input dimensions, sum the terms for `j = 0`, `1`, and `2`.
- Compute once with `D = 3`, then again with `D = 5`.
- The answer booklet appears to cover this under a different problem number, so review carefully.

## Solution Notes

problem numbering differs from the current textbook and needs human review.

- The result for three hidden units matches the seven-region figure.
- The five-hidden-unit result is checked against the candidate answer page.
