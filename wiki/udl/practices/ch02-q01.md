# Ch02 Q01: Least-Squares Gradients

## Prompt

Starting from the least-squares loss for the line model, derive the slope of the
loss with respect to the intercept and with respect to the slope parameter.

## What This Tests

Whether the loss is understood as a function of parameters, not just a number.

## Hints

- Treat every training pair as fixed.
- Differentiate one squared error term, then sum over all examples.
- The derivative with respect to the slope gets an extra factor of `x_i`.

## Solution Notes

Review needed against source and handwritten algebra.

## Self-Check Rubric

- The answer has one expression for each parameter.
- The expressions sum over all training examples.
- The slope-parameter expression includes the input values.
