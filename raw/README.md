# Raw Source Intake

`raw/` stores source intake material and manifests. Treat this layer as immutable evidence, not authored curriculum.

## Rules

- Record each source in a manifest before using it.
- Keep source files exactly as received when they are committed.
- Put corrected OCR, normalized Markdown, or extracted snippets in a clearly named derivative folder.
- Do not commit private or copyrighted full-text files unless the license and project policy allow it.
- Keep large local-only files under ignored `private/` directories.
