## Chapter 3

## Shallow neural networks

Chapter 2 introduced supervised learning using 1D linear regression. However, this model can only describe the input/output relationship as a line. This chapter introduces shallow neural networks. These describe piecewise linear functions and are expressive enough to approximate arbitrarily complex relationships between multi-dimensional inputs and outputs.

## 3.1 Neural network example

Shallow neural networks are functions $\mathbf{y}=\mathbf{f}[\mathbf{x},\boldsymbol{\phi}]$ with parameters $\boldsymbol{\phi}$ that map multivariate inputs $\mathbf{x}$ to multivariate outputs $\mathbf{y}$. We defer a full definition until section 3.4 and introduce the main ideas using an example network $\mathbf{f}[x,\boldsymbol{\phi}]$ that maps a scalar input $x$ to a scalar output $y$ and has ten parameters $\boldsymbol{\phi}=\lbrace \phi\_0,\phi\_1,\phi\_2,\phi\_3,\theta\_{10},\theta\_{11},\theta\_{20},\theta\_{21},\theta\_{30},\theta\_{31}\rbrace$:

$$
\begin{aligned}
y &= \mathbf{f}[x,\boldsymbol{\phi}]\\
&= \phi_0 + \phi_1 a[\theta_{10}+\theta_{11}x] + \phi_2 a[\theta_{20}+\theta_{21}x] + \phi_3 a[\theta_{30}+\theta_{31}x].
\end{aligned} \quad (3.1)
$$

We can break down this calculation into three parts: first, we compute three linear functions of the input data $(\theta\_{10}+\theta\_{11}x,\theta\_{20}+\theta\_{21}x,\text{ and }\theta\_{30}+\theta\_{31}x)$. Second, we pass the three results through an activation function $a[\bullet]$. Finally, we weight these resulting activations with $\phi\_1$, $\phi\_2$, and $\phi\_3$, sum them, and add an offset $\phi\_0$.

To complete the description, we must define the activation function $a[\bullet]$. There are many possibilities, but the most common choice is the rectified linear unit or ReLU:

$$
\begin{aligned}
a[z] &= \mathrm{ReLU}[z] \\
&= \begin{cases}
0 & z<0\\
z & z\geq 0
\end{cases}.
\end{aligned} \quad (3.2)
$$

This returns the input when it is positive and zero otherwise (figure 3.1).
