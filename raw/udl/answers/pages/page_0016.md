Deep neural networks
Figure 4.3 Hidden unit activations for problem 4.8. a) First hidden unit has a
joint at position x = 1/6 and a slope of one in the active region. b) Second hidden
unit has a joint at position x = 2/6 and a slope of one in the active region. c)
Third hidden unit has a joint at position x = 4/6 and a slope of minus one in the
active region.
Answer
Using the formula from problem 4.10, the shallow network has 3×95+1 = 286 parameters,
and the second network has 3 × 5 + 1 + (9 × 5 × 6) = 286 parameters. They are both the
same.
The shallow network can create 96 regions; since there is just one input, each hidden
unit creates one joint, for a total of 95 joints separating 96 linear regions. The number
of linear regions for the deep network is given by equation 4.17 and is 60,466,176.
In principle, the shallow network will be faster to run on modern hardware as the
computation is more parallel.
