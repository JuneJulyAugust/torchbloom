Problem 10.16 Consider the AlexNet network (figure 10.16 in book). How many parameters are
used in each convolutional and fully connected layer? What is the total number of parameters?
Answer
• Between the image and first layer, there are 3 × 96 × 11 × 11 = 34, 848 weights and 96
biases.
• Between the first layer and second layer, there are 96 × 256 × 5 × 5 = 614, 400 weights 256
biases
• Between the second layer and third layer, there are 256 × 384 × 3 × 3 = 884, 736 weights
and 384 biases.
• Between the third layer and fourth layer, there are 384 × 384 × 3 × 3 = 1, 327, 104 weights
and 384 biases
• Between the fourth layer and fifth layer, there are 384 × 256 × 3 × 3 = 884, 736 weights
and 256 biases
• At the end of the last convolutional layer, the representation is halved in size by the
maxpool operation. Rounding, down, it now has size 6 × 6 × 256 = 9, 216. So there are
9, 216 × 4096 = 37, 748, 736 weights and 4096 biases
• Between the next two fully connected layers, there are 4096 × 4096 = 16, 777, 216 weights
and 4096 biases.
• Between the last two fully connected layers, there are 4096 × 1000 = 4, 096, 000 weights
and 1000 biases.
Summing all these together, we get the total number of parameters is 34, 848 + 96 + 614, 400 +
256 + 884, 736 + 384 + 1, 327, 104 + 384 + 884, 736 + 256 + 37, 748, 736 + 4096 + 16, 777, 216 +
4096 + 4, 096, 000 + 1000 = 62, 378, 344 parameters.
This is not quite what they claim in the paper. This might possibly be because of the way they
trained across multiple GPUs. If anyone sees a mistake in my calculations or can explain this,
then please get in touch.
