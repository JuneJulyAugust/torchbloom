Problem 10.19 Consider two hidden layers of size 224×224 with C1 and C2 channels, respec-
tively, connected by a 3×3 convolutional layer. Describe how to initialize the weights using He
initialization.
Answer
He initialization is based on the number of weights that contribute to each hidden unit. In this
case, there would be C1 × 3 × 3 weights contributing to each hidden unit, so we would initialize
with mean zero and a variance of 2/9C1.
