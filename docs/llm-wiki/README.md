# LLM Wiki Docs

This folder defines the source-grounded LLM/AI concept wiki for TorchBloom.

- [llm-wiki-design.md](llm-wiki-design.md) - concept model, page schema, and build sequence.
- Raw source intake lives in `raw/llm-wiki/`.
- Synthesized wiki pages live in `wiki/llm-wiki/`.

The LLM wiki is a learning substrate, not a dump of source text. Its job is to connect math, programming, machine learning, deep learning, AI Olympiad reasoning, and paper-inspired projects into reviewable concept pages.
